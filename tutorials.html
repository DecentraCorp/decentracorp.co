TUTORIALS

Instructions are for Ubuntu/Debian
  • To complete this tutorial you need:
  • an open terminal
  • a web browser with the Metamask extension installed

install parity: bash <(curl https://get.parity.io -L) -r stable

clone the DecentraCorpPoA repo: git clone https://github.com/DecentraCorp/DecentraCorpPoA.git

move into the DecentraCorp directory: cd DecentraCorpPoA

use Nano to setup a new password: sudo nano pass.pwd

save your password: ctrl + x then y then enter

run parity with your password to create a new public key hash: parity --config config.toml account new copy this hash for future use Note: ctrl + shift + C to copy in Nano

open up config.toml: sudo nano config.toml

remove the # symbol before the #unlock(line 32) and #engine_signer(line 36) variables

replace the current hash values of unlock(line 32) and engine_signer(line 36) with your public key hash: unlock = ["PasteYourPublicKeyHashHere"] engine_signer = "PasteYourPublicKeyHashHere" Note: ctrl + shift + V to paste in Nano

save your updated config.toml file: ctrl + x then y then enter

connect to the network: parity --config config.toml Your node will sync and begin to download the blockchain Keep this terminal open and running in the background

Start your web browser and open your Metamask extension. Change the Metamask network to localhost:8545.

Follow this link to access DecentraCorps dApp to sign up for a membership

current membership terms and benefits are posted on the dApp site, and are likely to change after we complete our beta testing phase

to disconnect from the network simply press ctrl + C

close your browser and metamask before disconnecting from the network

to connect in the future: first, move to the DecentraCorpPoA directiory via terminal. second, connect to parity parity --config config.toml third, open your browser and metamask extension to access the dApps

Instructions are for Raspian. Other Pi friendly distros should work, but have yet to be officially tested
  • To complete this tutorial you need:
  • Raspberry Pi(tested on 2b and 3b+)
  • three GPIO connectors, wires, and a way to connect them to the antenna
  • something metal that can be used as an antenna

Assemble and attach your antenna to the pi using GPIO pins 4, 17, and 27.

now power on the Pi and open the terminal

install web3: sudo pip3 install web3

install RPi.GPIO: sudo pip3 install RPi.GPIO

Clone the Chaos Miner repo into the home directory: git clone https://github.com/DecentraCorp/ChaosMiner.git

move into the Chaos Miner directory: cd ChaosMiner

setup a new password: sudo nano chaos.pwd

save your password: ctrl + xthen y then enter

Run parity with your password to create a new public key hash: ./parity --config chaos.toml account new copy this hash for future reference

Open up chaos.toml: sudo nano chaos.toml

Remove the # symbol before the #unlock(line 32) and #engine_signer(line 36) variables

Replace the current hash values of unlock(line 32) and engine_signer(line 36) with your public key hash: unlock = ["PasteYourPublicKeyHashHere"] engine_signer = "PasteYourPublicKeyHashHere"

Save the updated chaos.toml file in Nano: ctrl + x then y then enter

Connect to the network: ./parity --config chaos.toml Your node will sync and download the blockchain keep this terminal open and running in the background

Open a new terminal window or tab and open the ChaosMiner directory: cd ChaosMiner

Activate your Chaos Miner: ./Activate.py this will prompt you to enter the SAME username and password that you use for your DecentraCorp dApp membership

Execute MineChaos.py: ./MineChaos.py A series of about 19 random numbers should begin to appear line by line keep this terminal open and running in the background

to disconnect: first, close the terminal running MineChaos.py ctrl + z second, close the terminal connected to the network ctrl + c

to connect in the future: first, move to the ChaosMiner directiory via terminal. second, connect to parity parity --config chaos.toml third, open a new terminal and run the MineChaos.py script (also in the ChaosMiner directory) ./MineChaos.py

This tutorial requires you to have an existing node and membership Please see the DCPoA and Membership tutorial!
  • To complete this tutorial you need:
  • A Computer with parity node client installed
  • A parity node running and connected to the DecentraCorp Proof of Authority Network
  • A DecentraCorp Membership and enough NotioCoin to stake a NodeMiner(50NTC)
  • A browser with metamask connected to the local parity node
  • The username you entered when you created your Account

This step assumes you have just done the above and picks up with the assumption that your node is running

open a new terminal and navigate to your cloned DecentraCorpPoA repo with cd path/to/DecentraCorpPoA

Activate the replication: ./Activate.py

You will need the user name you chose during your Membership Creation.

The staked NotioCoin will be removed automatically from your account.

Activate the Embeded Program Mining Software: ./Activate.py

Thats it! You are now Mining NotioCoin from UseBlocks on the CryptoPatent Blockchain!

To set both the node and the EPMS into the background run the following commands in each terminal: Ctrl+z bg disown

After this you can safely close both terminals and they will continue running in the background.
